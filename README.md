## 📖 Overview

**Math2DL** is an open-source educational project that explains **Deep Learning through mathematics and animation**.

Instead of treating neural networks as black boxes, this repository builds every major deep learning architecture from **first principles**, using:

* 🎥 High-quality **Manim** animations
* 📐 Mathematical derivations
* 🧠 Geometric intuition
* 📜 Formal proofs
* 💻 Minimal PyTorch implementations

The goal is to help students, engineers, and researchers understand **why deep learning works**, not just **how to use it**.

---

## 🎯 Mission

> **Bridge the gap between mathematical theory and practical deep learning through visual explanations.**

Every topic follows the same philosophy:

```
Mathematics
      ↓
Geometric Intuition
      ↓

      ↓
Implementation
      ↓
Real Deep Learning Model
```

# 🚀 Project Philosophy

Math2DL is built on four principles.

### ① Mathematical Rigor

Every concept is derived from its mathematical foundations.

---

### ② Visual Intuition

If a concept can be visualized, it should be animated.

---

### ③ Minimal Implementations

Every animation is accompanied by clean, readable code.

---

### ④ Research-Oriented Learning

Whenever possible, explanations reference the original research papers that introduced the ideas.
